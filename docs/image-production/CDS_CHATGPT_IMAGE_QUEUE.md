## Global Generation Rules

- **Visual language:** Premium cinematic cyber-gothic dark symphonic metal photography. Void black, structural violet, restrained cyan, pulse crimson, and witness gold. Human-led visual authorship, realistic materials, controlled signal distortion, no generic neon-city cyberpunk.
- **Public presentation:** All human characters are clearly adult and wear structured, fully opaque public-facing stagewear. Character sheets define identity, not mandatory exposure or prompt-unsafe detail.
- **Text:** Do not render titles, captions, logos, watermarks, fake UI words, or random typography. Website text remains HTML/CSS.
- **Groups:** Use only the sheets for characters present. Preserve fixed left-to-right order, distinct faces, hair, body classes, materials, and clothing languages. Never duplicate or merge characters.
- **Mobile:** Every -M item is a true vertical recomposition. Do not crop the desktop image.
